What is this?

This is a series of Org documents which contain all the code for all my configurations. To make use of it, open each file in Emacs. Then tangle all the code to the right places with: 

M-x org-babel-tangle

I will eventually make this easier.

What's actually here?

This repo contains literate configurations for:

  • Emacs
  • Windows 10
  • Vim
  • Arch Linux (btw)
  • Remote Server
